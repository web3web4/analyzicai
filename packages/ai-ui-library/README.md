# @web3web4/ai-ui-library

AI-specific UI components for analysis applications.

## What's Inside

AI-focused components:

- **AnalysisCard** - Display AI analysis results with scores
- **ApiKeyInput** - API key entry for AI providers
- **ProviderSelector** - Choose AI providers (OpenAI, Gemini, Claude)
- **ScoreBadge** - AI score display (0-100)
- **RetryPanel** - Retry failed AI analyses
- **BusinessSectorSelector** - Business context for AI analysis
- **InputTypeToggle** - Screenshot vs URL input

## Usage

```typescript
import {
  AnalysisCard,
  ApiKeyInput,
  ProviderSelector,
  ScoreBadge,
} from "@web3web4/ai-ui-library";
```

## Domain

This package contains **AI-specific UI components** used in analysis workflows.

For platform infrastructure (auth, admin, general UI), see `@web3web4/shared-platform`.
